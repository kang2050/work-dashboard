Handoff 助手接力 · 每张项目卡直接开工

每张项目卡底部都带 Claude 和 Codex 两个按钮。点一下，ProjectAssistantHandoffService 扫 ~/.claude/projects/ + ~/.codex/sessions/ 找这个项目最近的会话，写出 .memory/assistant-handoff.md（最近 3 条提问 + 2 条回复摘要、当前 Git 分支、dirty 文件清单），再用 AppleScript 打开一个 Terminal 窗口，把项目路径复制到剪贴板、拉起 CLI、让助手自己读这份 handoff。内部分享版通过 AppFlavor.supportsAssistantLaunch 直接禁用，按钮都不渲染。

下载与安装

分享版免费下载，无需注册。app 只读写你本机的文件，不上传任何项目内容。

⬇ 下载最新分享版 .zip 当前 v1.7.1 · 约 8MB · 需要 macOS 14 (Sonoma) 及以上

安装步骤

  1. 下载 zip，双击解压得到 工作看板.app，拖进「应用程序」文件夹
  2. 首次打开：右键点击 app → 「打开」→ 再点「打开」（分享版未做 Apple 公证，直接双击会被 Gatekeeper 拦一次，右键打开只需一次）
  3. 首次启动会引导你选择要扫描的项目根目录（如 ~/Projects），之后它就照着你的目录干活

分享版包含什么

分享版 = 本地项目看板的完整核心：目录扫描、项目卡片墙、Git 状态、.project.json 元数据、日志回顾、洞察与规则手册。 页面里演示的「对话立项 / AI 助手接力 / 运维自动化」属于作者自用的完整版，依赖本机的 Claude/Codex CLI 与私有基础设施，暂不随分享版提供。
