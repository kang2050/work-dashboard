# 工作看板 · WorkDashboard

本地优先的 macOS 项目工作台：扫描你本机的项目目录，把项目卡片、Git 状态、`.project.json` 元数据、工作日志、洞察和规则手册收进一张桌面看板。**项目内容不上传，没有云端，没有账号。**

📖 完整介绍：https://kang2050.github.io/work-dashboard/

## 下载

👉 [Releases 页面](https://github.com/kang2050/work-dashboard/releases/latest) 下载最新 `工作看板-vX.X.X-share.zip`

- 系统要求：macOS 14 (Sonoma) 及以上，Apple Silicon
- 安装：解压 → 拖入「应用程序」→ **首次右键点击 app 选「打开」**（未做 Apple 公证，直接双击会被 Gatekeeper 拦，右键打开一次即可）
- 首次启动会引导选择要扫描的项目根目录

## 分享版说明

分享版包含本地看板全部核心能力（目录扫描 / 卡片墙 / Git 状态 / 元数据 / 日志 / 洞察 / 规则手册）。介绍页中的「对话立项、AI 助手接力、运维自动化」依赖作者本机的 CLI 与私有基础设施，属于完整版，暂不随分享版提供。
